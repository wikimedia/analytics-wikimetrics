The following is a list of available metrics:

threshold

measures whether a user reached a given level of activity within a specific amount of time.

survival

measures a user's retention. A user is considered as "retained" or "surviving" if they continue editing after a time since a reference event (typically: account registration time).

edit rate

measures the rate of an editor's contributions during a given timespan

revert rate

measures the quality of a user's contributions expressed as the proportion of reverted edits within a given amount of time

time to threshold

measures how long it takes for a user to reach a given level of activity

block rate

represents whether the user has been blocked within a given amount of time

bytes added

measures the amount of content added, removed or modified by a user within a given amount of time

namespace edits

measures the breakdown of an editor's contribution by namespace
